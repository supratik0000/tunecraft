Where the music comes from

Tunecraft is built to be self-hostable and legally safe to share — nothing in it is licensed from a third-party catalogue.

Audio

All real-audio tracks in Tunecraft are SoundHelix demo songs — algorithmically generated MP3s that the SoundHelix project explicitly publishes for free use. Tracks without an audio file fall back to the in-browser Web Audio synthesizer included in this app.

Source: soundhelix.com

Track metadata

Every track title, artist name, album name, and emoji in the catalogue is original to this app. None of them refers to a real commercial release. The Hindi and Bengali entries use romanised originals, not the titles of any existing songs.

Code & design

Tunecraft was built as a self-contained, original music-player project — see the README for the open-source MIT license, full source layout, and deployment instructions.

Use your own music

To swap any track for your own MP3: drop the file into public/audio/, then change the audio: path on that track in server/db/catalog.js. The next deploy picks it up.
